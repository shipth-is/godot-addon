# ShipThis Godot Editor Addon

Godot editor addon to build and publish iOS and Android games using [ShipThis](https://github.com/shipth-is/cli).

- [x] Authentication - https://github.com/shipth-is/godot-addon/pull/2

- [X] Show the build output - https://github.com/shipth-is/godot-addon/pull/8
- [ ] Setup an Android game - https://github.com/shipth-is/godot-addon/issues/7
